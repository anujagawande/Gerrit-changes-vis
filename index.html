Gerrit is an open source, code version control tool. A 'Change' in Gerrit represents a single commit under review. Gerrit does not have built in data analysis charts like Git.
NDN(Named Data Networking) is a proposed Future Internet architecture research project.
In these 3 charts, 'Changes' in NDN Gerrit project repository has been analyzed.

This chart displays days taken for a change from the date it was created till the date it was submitted and merged in master branch.
Data displayed is from Jan 2019 - July 2020

Maximum number of days(29) were taken by Change - face+fw: Store and return PitToken from downstream submitted on 25 Sep 2019
33 Changes took 1 day for submission. Other Changes don't have a noticable trend in time taken for submission during this time period.
